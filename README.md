# Analysis-of-Road-Accidents
Built an interactive Power BI dashboard to analyze road accident data by performing data cleaning, transformation, and modeling using Power Query. Created DAX measures to track key metrics and visualized trends based on time, location, and contributing factors to generate actionable insights.
Data Analytics Project – Power BI Dashboard
Overview

This project focuses on analyzing a dataset using Power BI to generate meaningful insights through data visualization. The entire workflow, including data loading, cleaning, transformation, and dashboard creation, was performed within Power BI.

The objective is to present key trends and patterns in a clear, interactive, and business-friendly format.

Dataset
The dataset contains structured data used for analytical purposes
Includes both categorical and numerical features
Data was cleaned and transformed within Power BI using Power Query
Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Excel/CSV (Data Source)
